# Motorcycle Riding Guide Website

## Overview

The Motorcycle Riding Guide is a multi-page educational website designed to help new and intermediate riders learn about motorcycle safety, riding techniques, maintenance, riding gear, and motorcycle types.

This project was developed as part of Arizona State University's Computer Information Systems curriculum and demonstrates front-end web development principles and user-focused website design.

## Features

- Multi-page website architecture
- Responsive design for desktop and mobile devices
- Interactive contact form using JavaScript
- Educational content organization
- Navigation system across multiple pages
- Embedded multimedia content
- Data tables and structured information displays

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Skills Demonstrated

- Front-End Web Development
- Website Design
- Responsive Design
- User Experience (UX)
- Information Architecture
- Content Organization
- JavaScript Event Handling
